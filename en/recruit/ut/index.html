About the Project

The Xiangshan Project is a high-performance RISC-V processor core open-source project based on Chisel. As a core infrastructure component, the current UT (Unit Test) verification module requires a sustainable developer ecosystem. We are now publicly recruiting the first batch of module maintainers to collaboratively enhance the verification framework. Click here to fill out the application form. After submission, join the QQ group (Group ID: 600480230) to contact us.

Basic Requirements

UT module maintainers must possess a high-level understanding of at least one module to be maintained. For details on available modules, refer to the Progress Overview. Core Responsibilities

1. Documentation Updates and Maintenance

Continuously update and maintain design documentation based on existing knowledge and evolving Xiangshan source code, ensuring accuracy in reflecting the latest module designs and verification processes to guide future developers.

2. Verification Framework Maintenance

Learn the verification framework used by the UnityChipForXiangShan project, optimize the verification environment, and improve module descriptions to ensure verification accuracy and efficiency.

3. Open-Source Crowdsourcing

Publish prepared modules on the “UnityChip” platform, lead crowdsourced verification efforts, and provide technical support.
