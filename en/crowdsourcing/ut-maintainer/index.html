About This Project

The Xiangshan Open Source Project is a high-performance RISC-V processor open-source initiative launched by the Institute of Computing Technology, Chinese Academy of Sciences. The current UT (Unit Test) verification module serves as the core infrastructure of the project, requiring the establishment of a sustainable developer ecosystem. This program is publicly recruiting the first batch of module maintainers to collaboratively enhance the verification system. Click here to fill out the application form. After completing the application, please join the QQ group (Group ID: 600480230) for further communication.

Core Responsibilities

1. Module Learning

Through in-depth study of the verification documents provided by designers and the latest Xiangshan source code, familiarize yourself with the module’s functionality, understand the tasks it performs, grasp the signal names and their definitions, and keep up with the latest updates in the Xiangshan design source code. For currently available modules and their descriptions, please refer to the Progress Overview.

2. Documentation Compilation

Based on existing knowledge, compile and maintain design documents to ensure they accurately reflect the module’s design and verification processes, providing clear guidance for subsequent developers.

3. Verification Environment Setup

Learn and master the verification framework used by the UnityChipForXiangShan project, set up and optimize the verification environment to ensure the accuracy and efficiency of module verification.

4. Open Source Crowdsourcing

Publish the prepared modules on the Wanzhong Yixin website, lead the crowdsourced verification efforts, and provide technical support.

Reward Description

Participants will receive one-on-one code review opportunities with core Xiangshan developers, along with generous monetary rewards. Outstanding performers will have the chance to secure additional online/offline internship opportunities.
